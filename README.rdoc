== Recipe Finder

Tool created as part of Johns Hopkins "Ruby on Rails: An Introduction" Coursera course.

Visit http://recipefinder58290387.herokuapp.com to see it in action.

You may specify a keyword to search recipes on www.food2fork.com for by specifying a search term.
For example, http://recipefinder58290387.herokuapp.com/?search=bread.

Also check out README.md to see the full Functional Requirement of the project and the gems to be installed.
